Дипломный проект в команде «FakeNFT»

Приложение помогает пользователям просматривать и покупать NFT (Non-Fungible Token). Функционал покупки иммитируется с помощью мокового сервера. Просмотр коллекций NFT. Просмотр и покупка NFT (иммитируется). Просмотр рейтинга пользователей.

  • Сверстал приложение кодом.
  • Использовал UICollectionView, UITableView, TabBarController, NavigationController, WebViewController и GCD.
  • Внедрил тёмную тему.

Мобильное приложение «Tracker»

github.com/prostokot14/Tracker

Приложение помогает пользователям формировать полезные привычки и контролировать их выполнение. Контроль привычек по дням недели. Просмотр прогресса по привычкам.

  • Сверстал проект кодом на архитектуре MVVM.
  • Использовал HTTP-запросы и REST.
  • Хранение данных осуществил через CoreData.
  • Написал Snapshot-тесты и внедрил аналитику Yandex Mobile Metrica.

Мобильное приложение «ImageFeed»

github.com/prostokot14/ImageFeed

Многостраничное приложение предназначено для просмотра изображений через API Unsplash. Просмотр бесконечной ленты картинок из Unsplash Editorial. Просмотр краткой информации из профиля пользователя.

  • Создал удобный и интуитивно понятный интерфейс для приложения, используя storyboard.
  • Реализовал авторизацию пользователя через OAuth 2.0 Unsplash API.
  • Написал Unit и UI-тесты
  • Работал с Keychain через KeychainWrapper
  • Интегрировал Kingficher для загрузки фото

Мобильное приложение «MovieQuiz»

github.com/prostokot14/MovieQuiz-ios

Одностраничное приложение с квизами о фильмах из топ-250 рейтинга и самых популярных фильмов IMDb. Пользователь приложения последовательно отвечает на вопросы о рейтинге фильма. По итогам каждого раунда показывается статистика о количестве правильных ответов и лучших результатах пользователя. Цель — правильно ответить на все 10 вопросов раунда.

  • Реализовал приложение через паттерн Фабрика.
  • Написал Unit и UI-тесты
  • Использовал HTTP-запросы и REST.
